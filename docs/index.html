Utility First

The strongest design path is a token that pays for useful services instead of relying on speculation. AUT is structured around practical platform access.

AI automation credits

Use tokens to buy task credits for agents, workflows, API calls, and business automation.

Staked quota

Stake tokens to unlock API quota, rate limits, premium automation tiers, or platform discounts.

Marketplace access

Use tokens for listings, templates, integrations, creator tools, and verified agent workflows.

Verified Locally

The token and staking program are functional on a local Solana validator and on Solana devnet. Devnet staking smoke testing has verified stake, claim, and unstake behavior.

Stakable By Design

Staked principal is held in on-chain PDA-owned vaults. The host machine can run validator and monitoring infrastructure, but it does not custody user funds.

Stake

Users deposit tokens into a program-controlled vault.

Claim

Rewards accrue according to the configured rate and are paid from a funded reward vault.

Unstake

Users withdraw principal according to published staking rules.

Free Publishing Path

The cheapest public launch path is a static website on GitHub Pages, mirrored to Cloudflare Pages if desired, and optionally pinned to IPFS for an immutable snapshot.

AUT is a development prototype. It is not a public mainnet token, investment product, stablecoin, or promise of profit. Mainnet deployment requires dependency hardening, legal review, security review, treasury controls, validator operations rehearsal, and launch approval.
